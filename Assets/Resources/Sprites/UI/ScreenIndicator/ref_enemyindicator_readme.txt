-인디케이터가 나타날 때는 1~5프레임의 애니메이션, 인디케이터가 사라질 때는 5~7프레임의 애니메이션을 사용합니다.
-모든 프레임의 프레임속도는 0.064입니다.
-평소에는(나타나거나 사라지지 않을때) 5프레임만을 사용합니다.
-인디케이터가 나타나는 조건을 체크하기 위해 화면을 둘러싸는 감지 영역이 존재합니다. 감지영역의 형태는 가로의 두께가 9픽셀, 세로의 두께가 16픽셀인 속이 빈 직사각형 형태입니다. (참고 이미지: ref_detectingarea.png)
-화면 안쪽의 적(플레이어가 볼 수 있는 적)의 피봇이 감지 영역에 닿으면 인디케이터가 나타납니다.
-화면 바깥쪽의 적(플레이어가 볼 수 없는 적)의 피봇이 감지 영역에 닿으면 인디케이터가 사라집니다.
-인디케이터의 방향은 아카네의 피봇과 적의 피봇을 잇는 벡터의 단위벡터입니다.
-인디케이터의 스프라이트 중심은 화면 테두리에서 32픽셀 떨어져 있습니다.
