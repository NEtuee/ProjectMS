-대시 포인트(이하 AP)는 총 4개입니다.
-AP의 위치는 첨부한 이미지를 참고해주세요. (ref_apposition.png 참고) (체력바 크기와 위치가 바꼈으니 그것도 적용해주세요) (체력바 위치는 화면 왼쪽 끝에 체력바 왼쪽 끝 붙여주시고 화면 아래 끝에서 12픽셀 떨어져 있습니다)
-왼쪽에 있는 AP가 오른쪽의 AP보다 레이어 순서가 높습니다.(왼쪽으로 갈수록 z-index가 더 높음)
-AP 애니메이션이 적용되는 상황에는 4가지가 있습니다. (ref_apanimationtree.png 참고)
	1. 아카네의 혈섬(공격)이 적에게 적중하지 않아 AP가 소멸 (breaking, 5프레임)
	2. 시간이 지나 소멸됐던 AP가 자동적으로 회복 (autocharging, 6프레임)
	3. 혈섬(공격)을 적중시켜 소멸됐던 AP가 회복 (absorbing, 8프레임)
	4. 혈섬(공격)을 적중시켜 AP가 강조 (accenting, 프레임) (아직없음ㅎ)
-각 애니메이션의 프레임별 속도는 다음과 같습니다.
	breaking : 0.032 0.032 0.064 0.064 0.128
	autocharging : 0.064 0.064 0.064 0.064 0.072 0.072
	absorbing : 0.048 0.048 0.072 0.072 0.096 0.096 0.096 0.096
	accenting : 

-가득찬 AP는 full.png를 사용합니다.
-빈 AP는 breaked.png를 사용합니다.
-breaked.png와 각 애니메이션은 A, B, C, D의 4가지 바리에이션을 가지고 있습니다.
-full.png에서 다른 애니메이션을 출력할 때, 4가지 바리에이션 중 하나를 랜덤으로 지정해 재생합니다.
-breaked.png에서 다른 애니메이션을 출력할 때, breaked.png의 바리에이션과 같은 알파벳을 가진 애니메이션을 재생합니다.
-애니메이션 진행 도중 다른 상황이 발생하면, 진행중인 애니메이션을 중단하고 새 애니메이션을 처음부터 재생합니다.
